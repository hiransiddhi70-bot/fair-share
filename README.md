### 🧠 How It Works - Algorithm

Uses **Greedy Algorithm** to minimize cash flow:

1. Calculate total spent & average per person
2. Split into lenders (paid extra) & borrowers (paid less)
3. Always settle highest lender with highest borrower first
4. Result: Minimum number of transactions needed ✅

**Example:**  
Input: Siddhi ₹1000, komal ₹500, arju ₹0, Total ₹1500, Avg ₹500  
Output: `arju pays Siddhi ₹500` - Only 1 transaction instead of 3!
